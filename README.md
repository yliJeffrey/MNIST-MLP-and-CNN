# MNIST-MLP-and-CNN
MLP and CNN with different structures are trained separately on MNIST dataset.<br>
Best models are saved in the folder bestModel; Images of each network are saved in folder network.
<br>

Python 3.11.8 <br>
Tensorflow 2.15.0 <br>
numpy 1.26.4 <br>
keras 2.15.0 <br>
cuda 12.5 <br>
gpu: RTX3060 <br>
